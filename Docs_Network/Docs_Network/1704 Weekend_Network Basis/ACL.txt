ACL(Access Control List)
  - 접근 제어 목록
  - 네트워크에 접속하는 트래픽에 대한 검사를 통해
    접근을 제어 하는 기술
  - 트래픽을 검사할 때에는 IP 주소와 Port 번호를 확인
    하여 허가된 IP 주소와 Port 번호에 대해서만 접근을
    허용 한다.

표준(Standard) ACL
  - 출발지 IP 주소만을 식별(검사)
  - 1 ~ 99번 까지 사용(1300 ~ 1999)

확장(Extend) ACL
  - 출발지/목적지 IP 주소
    Protocol Type : IP, ICMP, EIGRP, OSPF, UDP, TCP
    출발지/목적지 Port Number
  - 100 ~ 199번 까지 사용(2000 ~ 2699)

Named ACL
  - 표준/확장 사용
  - ACL 번호를 사용하는 대신 이름을 부여하여 사용

Inbound/Outbound
  - 트래픽을 식별 할 때 특정 인터페이스로 들어오는(Inbound) 데이터와
    특정 인터페이스로 나가는(Outbound) 데이터에 대해 개별 적으로 식별을
    하게 된다.
