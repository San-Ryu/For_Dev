■ Oracle VM Virtual 환경설정
	환경설정 >> 기본 머신 폴더 >> 기타
	(D드라이브로 디폴트 경로 변경 (C에 저장하면 User 변경 등의 상황에서 파일을 다시 열 수 없는 상황이 생김))
	버전 확인 : 도움말 > Virtual Box 정보
	버전 업데이트 : 도움말 > 업데이트 찾기
	또는 네이버 자료실 최신버전 다운 (초기설정 그대로 업데이트 진행 가능)
